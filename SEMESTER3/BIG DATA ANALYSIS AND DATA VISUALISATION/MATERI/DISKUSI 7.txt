Persamaan
Keduanya merupakan bagian dari ilmu data (data science) dan digunakan untuk menganalisis data dalam jumlah besar demi menemukan pola atau menghasilkan prediksi.
Sama-sama menggunakan teknik statistik, matematika, dan algoritma untuk mengenali pola dalam data.
Seringkali, algoritma machine learning digunakan dalam proses data mining untuk meningkatkan akurasi hasil analisis.
Tujuan akhirnya adalah membantu pengambilan keputusan berbasis data dengan hasil yang lebih baik.

perbedaan
Data mining
Data mining berfokus pada penemuan pola atau hubungan tersembunyi yang sebelumnya tidak diketahui dari kumpulan data besar
Tujuan Data mining utamanya adalah mengubah data mentah menjadi informasi yang mudah dimengerti dan bermanfaat untuk pengambilan keputusan
Pada data mining, prosesnya biasanya dimulai dengan data historis, lalu dilakukan analisis untuk menemukan pola. 
Prosesnya sering membutuhkan intervensi manusia dalam memilih data, menentukan teknik analisis, hingga interpretasi hasil

Machine learning
Machine learning bertujuan membangun model yang mampu belajar dari data yang ada untuk membuat prediksi atau keputusan otomatis tanpa instruksi eksplisit dari manusia.
Fokus utamanya adalah pada proses pembelajaran mesin secara mandiri agar performa prediksi semakin baik seiring waktu
Pada machine learning, prosesnya melibatkan pelatihan (training) model pada data (baik berlabel maupun tidak), lalu model tersebut digunakan untuk memprediksi data baru. 
Proses pembelajaran dan penyempurnaan model berlangsung otomatis, dengan minim intervensi manusia setelah tahap awal
